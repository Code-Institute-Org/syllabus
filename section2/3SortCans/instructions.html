  • Control loop asideOpen or Close

    The controller code is generally divided into two or three phases:

    1. The initialisation phase consists of the initialisation and start-up.
    2. The execution phase is where the robot performs actions until it is told to turn off. This step is where the control loop code is.
    3. The cleanup phase includes using the motors to move to a rest position and deactivating the sensors before shutdown.

    Let us look at an example to illustrate this:

    1. The robot controller in this topic:
      1. Initialises the arm by:
        • Enabling sensors
        • Rotating the joints to put the robot in its starting position.
      2. The control loop consists of:
        • Waiting for a can
        • Grabbing the can
        • Putting the can in a box.
        • Rotating the arm back to the waiting position.
      3. The cleanup phase is:
        • Rotating the arm back to the side
        • Closing log files
        • Deleting variable values
        • Shutting down the sensors and motors

    Within this simulation, we never exit the control loop, so the cleanup phase is not used. However, in a real-world factory, You would use the cleanup phase when powering down the arm.

    We will write the code required within the control loop in the next task. This code needs to recognise the colour of a can when the arm grabs it.

  • Solution for (Get the colour of the can)Open or Close 
    def get_colour_of_can():
    camerapixels = camera.getImage()
    red = Camera.imageGetRed(camerapixels, camera.getWidth(), 0, 0)
    green = Camera.imageGetGreen(camerapixels, camera.getWidth(), 0, 0)
    blue = Camera.imageGetBlue(camerapixels, camera.getWidth(), 0, 0)

    largest = max(red, green, blue)
    colour = ""
    if red == largest:
        colour = 'red'
    elif green == largest:
        colour = 'green'
    else:
        colour = 'blue'

    print(colour)
    return colour
